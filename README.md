# majestic-law-db

База законов всех серверов Majestic RP в JSON. Скрапится с официального форума, нормализуется и размечается: у каждой статьи есть короткий титул, поисковые теги и разобранные санкции (штраф, срок, звёзды розыска, залог, флаги).

Используется приложением [Majestic-Law-Overlay](https://github.com/h3xbleed/Majestic-Law-Overlay) и веб-версией. Можно брать для своих некоммерческих поделок — см. лицензию ниже.

## Файлы

```
db/
  <Server>.json    # 19 серверов: Atlanta ... Washington
  index.json       # сводка: даты скрапа, количество документов и статей
changelog/
  YYYY-MM-DD.md    # человекочитаемый ченджлог
  YYYY-MM-DD.json  # он же в структурном виде
```

Дата актуальности — поле `scraped_at` внутри каждого файла и в `index.json`. История изменений — коммиты этого репо плюс ченджлоги.

## Схема

```jsonc
{
  "server": "Detroit",
  "scraped_at": "2026-07-06T…",
  "documents": [{
    "id": "2000854",              // id темы на форуме
    "title": "Уголовный Кодекс",
    "type": "кодекс",             // кодекс | закон | конституция | прочее
    "section": "Кодексы",
    "url": "https://forum.majestic-rp.ru/threads/…",
    "article_count": 167,
    "chapters": [{
      "label": "Глава", "num": "6", "title": "Преступления против жизни…",
      "articles": [{
        "num": "6.3",
        "title": "Тяжкое убийство двух и более лиц",   // краткий титул
        "tags": ["убийство", "мокруха"],                // поисковые синонимы
        "text": "…полный текст статьи…",
        "flags": ["A", "CR"],
        "jur": "LSPD",                                  // юрисдикция, если выводима
        "sanctions": {
          "prison_max": 50, "prison_unit": "месяц",
          "wanted_stars": 5, "bail_denied": true,
          "fine_min": 15000, "fine_max": 15000,         // где есть штраф
          "license_revoke": true                        // и т.п.
        }
      }]
    }]
  }]
}
```
